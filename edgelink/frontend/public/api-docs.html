ðŸš¦ Quick Start

Get started with EdgeLink API in seconds:

1. Create an account

curl -X POST https://edgelink-production.quoteviral.workers.dev/auth/signup \
  -H "Content-Type: application/json" \
  -d '{
    "email": "you@example.com",
    "password": "securePassword123",
    "plan": "free"
  }'

2. Create a short link

curl -X POST https://edgelink-production.quoteviral.workers.dev/api/shorten \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com/very-long-url",
    "custom_slug": "my-link"
  }'

3. Get analytics

curl -X GET https://edgelink-production.quoteviral.workers.dev/api/analytics/my-link?range=7d \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
